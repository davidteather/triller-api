## Getting Started

Run the following command to install it from PyPi
```sh
pip install TrillerAPI
```

Basic Quick Start Script
```py
import TrillerAPI
user = TrillerAPI.login("YOUR_TRILLER_USERNAME", "YOUR_TRILLER_PASSWORD")
trending = user.get_trending()
```
